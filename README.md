## Abstract

We propose **CARNet**, which addresses **Multi-view Consistency Collapse (MCC)**
across all three stages of the light field depth estimation pipeline:

- 🔹 **CSFM** – Consistency-Sensitive Feature Module with low/high-frequency adaptive encoding
- 🔹 **MCRM** – Multi-view Collapse Risk Map for targeted residual refinement  
- 🔹 **CPSO** – Collapse-Prioritized Supervision Objective with plug-in capability

---

## Results

Evaluated on **HCI 4D Benchmark** and **UrbanLF-Syn**.  
CARNet achieves state-of-the-art Bad-Pixel Rate (BPR) performance  
while maintaining competitive MSE.
